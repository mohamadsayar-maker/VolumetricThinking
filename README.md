# Volumetric Thinking

Landing page for *Volumetric Thinking*, a book on spatial experience design by Mo Sayyar.

The page is a single self-contained `index.html`: a real-time cloth simulation
(Three.js) drapes a "+" pattern over an invisible form to reveal the title,
followed by the book, framework, author, and print-interest sections.

- `index.html` — the entire site
- `VolThinkingBgAudio.mp3` — background music (must sit next to index.html)
- Author photo: upload `mo-sayyar.jpg` next to index.html

Shortcuts: `***` style panel · `M` mute · `D` dark theme · `E` export hero image · `1/2/3` jump to sections.
